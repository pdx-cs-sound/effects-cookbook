# Single-sample effects

*Chapter 3. Not yet written — see [Status & scope](status.md).*

Effects that keep no state: each output sample is a function of the current input sample
alone. Planned content: the volume effect, distortion, mu-law, bit crush.

Mu-law is "companding" in the codec sense — compressing before storage and expanding after.
The same word describes the level effects of [Chapter 5](compression.md); the two uses are
related but not the same, and this book keeps them apart.
