# Voice samples — the Limiting chapter, three ways

The same chapter rewritten in three candidate voices, so we can pick one (or set the dials
between them) before rewriting the whole book. Rules 1–9 of `STYLE.md` apply to all three;
what varies is warmth, person, and humor. None of them uses bold or italics to carry
meaning.

To keep the comparison about prose, each sample covers only the prose sections (definition,
motivation, mechanism, pitfalls). The parameter table, pseudocode, reference code, and
reference lists are voice-neutral and identical across versions, so they are omitted.

| File | Voice | Touchstone | Character |
|---|---|---|---|
| `limiting-warm.md` | Warm teacher | Asimov non-fiction | Talks to the reader; "you," contractions, unhurried; builds each idea as a small story |
| `limiting-textbook.md` | Textbook | Prince, *Understanding Deep Learning* | Impersonal "we"; definitional precision; numbered mechanism; no contractions |
| `limiting-essay.md` | Essayist-engineer | lcamtuf (Zalewski) | Plain and concrete; dry asides in parentheses; assumes intelligence, not background |

## Questions for review

1. Which voice should the book use? Or: which base, adjusted how? (E.g. "textbook, but
   allow contractions" or "essay, minus the humor.")
2. Person: "you" (warm), "we" (textbook), or mostly impersonal with occasional "you" (essay)?
3. Humor tolerance: the essay voice contains three dry asides. Too many? Zero?
4. Whichever wins, the current pages will be rewritten to it in one pass; the pseudocode,
   tables, and code stay as they are.
